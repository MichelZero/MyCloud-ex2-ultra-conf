Instalação do htop via Entware

Este guia detalha o processo de instalação do gerenciador de pacotes Entware e da ferramenta de monitoramento htop no WD My Cloud EX2 Ultra (Arquitetura ARMv7).

Atenção: O sistema de arquivos raiz do NAS é volátil. Siga os passos de persistência para não perder a instalação ao reiniciar.
01

Preparar Armazenamento Persistente

Como a raiz (/) é resetada no boot, instalaremos tudo no HD principal (sda2).

mkdir -p /mnt/HD/HD_a2/entware rm -rf /opt ln -sf /mnt/HD/HD_a2/entware /opt
02

Executar o Instalador Entware

Baixe e execute o script de instalação para a arquitetura ARMv7 (Armada 385).

wget -O - http://bin.entware.net/armv7sf-k3.2/installer/generic.sh | sh
03

Instalar o htop

Atualize a lista de repositórios e instale o pacote desejado.

/opt/bin/opkg update /opt/bin/opkg install htop
04

Dica de Ouro: Persistência (Pós-Reboot)

Sempre que o NAS reiniciar, o link /opt sumirá. Para restaurar o acesso rápido sem reinstalar tudo, execute este comando único:

ln -sf /mnt/HD/HD_a2/entware /opt && export PATH=$PATH:/opt/bin:/opt/sbin

Dica: Salve este comando em um bloco de notas no seu PC para rodar via SSH sempre que reiniciar o NAS.

Como iniciar o monitoramento

Após a instalação, basta digitar o comando abaixo no shell:

htop
